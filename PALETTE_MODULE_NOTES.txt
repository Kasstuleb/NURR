NURR Palette mode integration

Added files:

- css/palette-mode.css

Changed files:
- index.html
- js/app.js

Notes:
- Palette mode is added as a fifth rail item.
- CSS is scoped under .nurr-palette-* and .palette-mode-* so it can be edited later without changing base.css.
- palette-lab-mode.js exposes:
  window.PaletteMode
  window.PaletteControls
  window.PALETTE_DEFAULTS
  window.NURR_PALETTE_SEEDS
  window.NURR_PALETTE_ENGINE

- JSON
